Initialization

Recoil.init( :scope )
:scope
The initial scope of your application.

Transitions

Recoil.createTransition( :type, :viewName, :function )
:type
A string representing the type of transition. Options: [ intro, outro ]
:viewName
The name(s) of the view(s) that this transition will be applied to.
:function
A function that performs the transition tasks.

View Path

Recoil.setViewPath( :path )
:path
The base path where your views are stored.

Update Throttling

Recoil.setMaxUpdateFrequency( :frequency )
:frequency
The maximum frequency that the global update method will be allowed to run. Default: 50ms
