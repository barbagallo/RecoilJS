String Interpolation

#\{ :string }
:string
An expression that parses to a string.

Compose Binding

data-compose=":scope" data-view=":view"
:scope
An object that will act as the default scope for all contained elements.
:view
A string containing a valid view name or HTML filename.

Context Binding

data-context=":scope"
:scope
An object that will act as the default scope for all contained elements.

CSS Binding

data-css=":cssObject"
:cssObject
An object that will be passed into jQuery.fn.css().

Event Binding

data-[:eventType]=":expression"
:eventType
A string representing a valid JavaScript event type (ie. click).
:expression
An expression to be evaluated when the defined event occurs.

Iteration

data-for=":identifier in :set when :condition"
:identifier
An identifier that will be used to reference the current item being iterated over.
:set
An array of items that will be iterated over.
:condition
An expression that will be run against each item, determining whether or not the item should be included.

HTML Binding

data-html=":htmlString"
:htmlString
A string of HTML that will be inserted into the DOM element.

If Binding

data-if=":ifExpression"
:htmlString

Static Binding

data-static

Update Binding

data-update=":updateExpression"
:updateMethod
An expression that will be run each time the current DOM element is updated.

Value Binding

data-value=":value"
:value
A variable or function that will be bound to the DOM element.

Visible Binding

data-visible=":visibleExpression"
:visibleExpression
An expression that will be evaluated to determine whether or not to hide the current element.
