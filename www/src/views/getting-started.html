Dependencies

There is really only one dependency for using RecoilJS:

jQuery
http://code.jquery.com/jquery-1.9.1.min.js

However, if you want CoffeeScript syntax to be supported in your bindings, you will want to include CoffeeScript as well:

CoffeeScript
https://github.com/jashkenas/coffee-script/blob/master/extras/coffee-script.js

Including RecoilJS

First, you're going to want to download a copy of recoil.js. Once you have the JavaScript file, you can include it in your page one of two ways:

  1. Include the file with a standard script tag: <script src="recoil.js"></script>
  2. Include the file via AMD with RequireJS: require( [ "recoil" ], function () { ... } )

For the purpose of getting started, I'm going to assume the first method.

HTML Setup

Setting up your HTML file is incredibly easy in that technically, you just have to include the JS files and you're set; however, you can optionally specify the scope of your application with the data-app attribute. Here is a sample HTML file:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8" />
<title>RecoilJS Sample</title>
<script src="jquery.js"><script>
<script src="coffee-script.js"><script> <!-- optional -->
<script src="recoil.js"><script>
<script src="app.js"><script> <!-- your application -->
</head>
<body>
<div data-app>
...
</div>
</body>
</html>

In this example, the <div> tag represents the scope of your app, and all parsing will start here. This tag can contain HTML, or it can be empty and have content loaded in through an external view.

The next thing your app is going to need is a Controller. This can be any JavaScript object. You can bind this object using the following code:

var controller = {
title: "Hello, World!"
}
Recoil.init( "sample", controller )

As you can see, our Controller can be incredibly simple - in this case, consisting of only one title property. Let's add some HTML!

<div data-app="sample">
#{ '#{ title }' }
</div>

The title has now been rendered into the HTML through the use of String Interpolation.

That's it!
